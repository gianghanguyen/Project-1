## Bài tập lớn môn Project 1
## Giải trò chơi 8-puzzle
Web-app giải trò chơi 8-puzzle sử dụng các thuật toán tìm kiếm 
- A*
- Breadth First Search
- Depth First Search

##  Chạy server
```
cd src
python app.py
```

## Chạy client
```
cd app
npm start
```
